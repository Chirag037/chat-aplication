📘 Intern Roadmap: Chat Application
(Django + Vue.js)
🎯 Objective
Build a full-stack chat application using Django (backend) and Vue.js (frontend) within one
week.
The application should allow users to:
• Register and log in
• Create or join chat rooms
• Send and receive messages

🧭 Project Overview
Tech Stack
•
•
•
•

Backend: Django + Django REST Framework
Frontend: Vue.js
Database: SQLite (default)
Communication: REST APIs (Polling-based updates)

Must Follow
• Use Django ORM (no raw SQL unless necessary)
• Keep API responses consistent (JSON)
• Separate concerns (models, views, serializers)

Avoid
• Hardcoding values
• Mixing frontend and backend logic
• Overcomplicating the architecture
